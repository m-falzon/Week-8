To load html and other web related files

  1. Create a web directory outside in the Finder (outside of Xcode)
  2. Drag the web directory into Xcode
  3. This is the only time its vital that you click the Create folder references radio button

As you've imported the web directory into Xcode, any files or folders you add under web will automatically show up in Xcode, and, loading of relative assets (e.g. css files) will work like normal.
